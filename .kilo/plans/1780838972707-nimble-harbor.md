# Plan: Fix goreleaser and CI for sancho CLI

## Goal
Ensure the release workflow works correctly with goreleaser for Homebrew distribution.

## Verified Issues

### 1. goreleaser.yaml - Build Configuration
**Status:** Needs fix
- `binaries: - sancho` syntax is incorrect - should be `binary: sancho` (singular, not a list)
- `goarm: 6` is for ARM builds, not arm64 - should use `goarm64` if needed
- Default `GOAMD64` for amd64 is `v1`, which is fine

### 2. goreleaser.yaml - Homebrew Configuration  
**Status:** Needs fix
- `tap.name: homebrew-tap` will push to `marioaer/homebrew-tap` repo
- Formula will be created at `Formula/sancho.rb` in that repo
- `folder: ./homebrew-tap` is where formula is generated locally before push

### 3. CI Workflow - Go Version
**Status:** Critical fix needed
- `go-version: '1.26'` is invalid - Go stable versions are `1.21`, `1.22`, `1.23`
- Should use `go-version: '1.22'` or `go-version: '1.23'`
- This will cause CI to fail immediately

## Proposed Changes
